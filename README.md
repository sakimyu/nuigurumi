# マインクラフト　「ぬいぐるみが持てちゃう」リソースパック
nuigurumi はMinecraftの統合版でスポーンエッグの見た目をぬいぐるみのように変えるリソースパックです。

* 羊
* 豚
* ニワトリ
* ハチ
* カメ
